# 创意工坊介绍页面生成计划

## 目标
根据 `Faction Gear Customizer` (Faction Gear Modification) 的代码库分析结果，生成一份详细、吸引人且结构清晰的 Steam 创意工坊介绍页面。

## 计划步骤

1.  **撰写标题与简介**
    *   使用醒目的标题（如：Faction Gear Customizer - 全面自定义派系装备）。
    *   编写简短有力的简介，说明 Mod 的核心功能：允许玩家完全控制 NPC 派系单位的武器、服装和健康状态。

2.  **详细列出功能特性 (Features)**
    *   **深度自定义**: 强调可以修改武器、服装、物品材质、品质、颜色等。
    *   **可视化编辑器**: 介绍游戏内的 GUI，支持筛选、搜索、分类查看。
    *   **兵种细分**: 说明可以针对特定兵种（PawnKind）进行单独配置。
    *   **高级选项**: 提及生物编码 (Biocoding)、强制剥离 (Strip)、健康状态 (Hediffs/Bionics) 添加等高级功能。
    *   **预设管理**: 介绍预设保存、加载和复制粘贴功能，方便玩家分享或快速配置。

3.  **编写使用指南 (How to Use)**
    *   简述如何打开编辑器（通常在 Mod 设置或开发者模式中）。
    *   说明基本操作流程：选择派系 -> 选择兵种 -> 修改装备 -> 保存。

4.  **说明兼容性与依赖 (Compatibility & Requirements)**
    *   列出依赖项：Harmony。
    *   说明与其他 Mod 的兼容性（如 Vanilla Expanded 系列），以及是否支持中途加入存档。

5.  **FAQ 与 注意事项**
    *   解答常见问题（如：是否影响性能、是否支持 CE 等）。

6.  **致谢 (Credits)**
    *   列出作者 (yancy) 及其他贡献者。

## 输出格式
*   最终输出将是一个 Markdown 文件，可以直接复制到 Steam 创意工坊的描述框中。
*   包含适当的 Markdown 格式（标题、列表、粗体）以提高可读性。
