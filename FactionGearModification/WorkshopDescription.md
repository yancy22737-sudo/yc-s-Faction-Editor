# Faction Gear Customizer - 全面自定义派系装备

**厌倦了总是遇到拿着相同破烂武器的海盗？想要让帝国禁卫军真正全副武装？或者给某个部落派系全员装备高科技义体？**

**Faction Gear Customizer** 让这一切成为可能！本 Mod 允许你在游戏内实时深度定制任何派系、任何兵种（PawnKind）的装备、武器甚至健康状态。

> **🌟 核心理念：轻量、现代、易用**
> 我们致力于提供最流畅的用户体验。拥有现代化的 UI 设计，支持一键导出分享配置，支持按 Mod 来源筛选物品，让复杂的自定义变得简单直观。

---

## ✨ 核心功能

### 🎯 精确到兵种的控制
不仅仅是针对派系，你可以精确控制到具体的兵种类型。
*   想让 **帝国骑兵 (Empire Trooper)** 装备特定的动力甲？没问题。
*   想让 **海盗枪手 (Pirate Gunner)** 只使用特定 Mod 的枪械？完全可以。

### ⚔️ 深度武器定制
*   **强制装备**：指定兵种必须携带的武器。
*   **材质与品质**：设定武器的材质（如玻璃钢、铀）和品质（如极佳、传奇）。
*   **生物编码**：设置武器被生物编码的概率，防止玩家轻易缴获。
*   **样式与颜色**：自定义武器的视觉样式。
*   **智能冲突检测**：Mod 会自动检测远程武器与护盾腰带的冲突，确保单位逻辑正常。

### 🛡️ 服装与外观
*   **强制着装**：指定兵种必须穿戴的特定服装（头盔、护甲、衣物等）。
*   **剥离原有装备**：可选择移除该兵种原本随机生成的服装，确保统一着装。
*   **颜色定制**：为服装指定特定的颜色，打造统一的军团制服感。
*   **分层管理**：分别管理头饰、外衣、内衣和配件，互不冲突。

### 🏥 健康与义体
*   **添加健康状态**：给生成的单位添加特定的健康状态（Hediff）。
*   **义体植入**：可以强制给某些兵种安装仿生腿、战术眼或其他植入物。
*   **概率控制**：设定每种健康状态出现的概率，创造多样化的敌人（如 30% 概率携带增强义体）。

### 🛠️ 强大的游戏内编辑器
*   **现代可视化界面**：无需修改 XML 文件，所有操作都在游戏内通过现代化图形界面完成，直观易用。
*   **高级 Mod 筛选**：支持按 Mod 来源筛选物品（如只显示 "Vanilla Expanded" 系列装备），配合科技等级、伤害值等过滤器，快速定位目标装备。
*   **预设管理与分享**：保存你的配置为预设，支持导出分享给朋友，或在不同存档间无缝迁移。
*   **复制粘贴**：支持在不同兵种间复制配置，快速批量应用。

## 📖 如何使用

1.  进入游戏，打开 Mod 设置。
2.  找到 **Faction Gear Customizer** 的编辑器按钮。
3.  在左侧选择你想要修改的 **派系**。
4.  在中间列表选择具体的 **兵种**。
5.  在右侧面板配置该兵种的 **武器、服装** 或 **健康状态**。
6.  点击保存，配置即刻生效（对新生成的单位有效）。

## ⚠️ 常见问题

**Q: 兼容 CE 吗？**
A: 我测试下来没问题。

**Q: 支持 Mod 派系吗？**
A: 理论上是的。

**Q: 这会影响已经存在的单位吗？**
A: 不会。修改只会影响新生成的单位（如新刷新的袭击、援军或商队）。

**Q: 如果我卸载 Mod 会怎样？**
A: 只有 Mod 的配置数据会丢失，游戏存档通常是安全的，但建议在卸载前备份存档。
