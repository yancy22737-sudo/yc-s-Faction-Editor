 [h2]âš ï¸ FAQ[/h2]
 
 [b]Q: Is it compatible with Total Control? [/b]
 A: Yes.
 
 [b]Q: Is it compatible with CE? [/b]
 A: Testing. Currently, the pawn kinds can generate weapons and ammunition properly, and all stats are working fine.
 
 [b]Q: Does it support Mod factions? [/b]
 A: Theoretically yes.
 
 [b]Q: Will this affect existing units? [/b]
 A: No. Modifications only affect [h3]newly generated[/h3] units (such as newly spawned raids, reinforcements, or caravans).
 
 [b]Q: Can I add this mid-save? [/b]
 A: No problem.
 
 [b]Q: What happens if I uninstall the Mod? [/b]
 A: Only the Mod's configuration data will be lost. Game saves are usually safe, may show a few red errors, but won't affect normal gameplay.
